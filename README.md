📱 Drug Application – Android App

Μια εφαρμογή Android για την καταγραφή, παρακολούθηση και διαχείριση φαρμάκων.
Υλοποιήθηκε στο πλαίσιο του μαθήματος Ανάπτυξη Κινητών Εφαρμογών.

📌 Λειτουργίες

Προσθήκη, επεξεργασία και διαγραφή φαρμάκων.

Αποθήκευση στοιχείων σε SQLite (Room Database).

Προβολή όλων των φαρμάκων με RecyclerView.

Σήμανση “Received Today” με αυτόματη επαναφορά κάθε μέρα.

Αυτόματη διαχείριση του Active με βάση Start/End Date.

Background ενημερώσεις μέσω WorkManager.

Προβολή τοποθεσίας ιατρού μέσω Google Maps (όπου υποστηρίζεται).

🛠️ Τεχνολογίες

Java

Android Studio

Room Database (SQLite)

WorkManager

RecyclerView

Google Maps Intent

📂 Δομή έργου

src/ – όλος ο κώδικας και τα resources της εφαρμογής

app/build.gradle – ρυθμίσεις module

.gitignore – αποκλεισμός περιττών αρχείων

Δεν περιλαμβάνονται build folders, .gradle, local.properties

Sync Gradle
